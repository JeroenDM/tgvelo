Banden

Grip / weerstand tegen lekken / redundantie / breedte

De parameter 'banden' is een verzameling van eigenschappen die aan een band kunnen gelinkt worden en een deel van de kwaliteit van de band bepalen. Goede banden zijn essentieel voor een veilige fiets, maar de banden hebben natuurlijk op veel meer parameters invloed, waar in deze parameter geen rekening wordt gehouden (maar wel in de andere parameters).

Definitie

De score 'Banden' is een vergelijking met het maximaal haalbare. Een hoge score betekent:

  • De banden hebben een hoge weerstand tegen het prikken van een naald van 2mm doorsnede (lekke banden kunnen de stabiliteit van de fietser in gedrang brengen).
  • De banden hebben een hoge zijdelingse dynamische wrijvingskracht bij een verticale belasting van 200N (een hoge grip zorgt voor zekerder bochtengedrag.
  • De banden kunnen snel hersteld worden in geval van lek (banden op druk dragen bij tot een veilig weggedrag).
  • De banden zijn breed (nuttig om minder gemakkelijk in spleten te blijven steken).

Meten Grip

De band wordt op een slede geïnstalleerd en belast met gewicht steunende op een gestandaardiseerde ondergrond (in het geval van TGVelo een ruwe terrastegel). Gewicht wordt toegevoegd tot de band met een kracht van 200 N op de ondergrond drukt. De slede met band kan zijwaarts worden voortgetrokken met een katrol die aan een vast object wordt gemonteerd. Tussen de katrol en de slede wordt een krachtcel gemonteerd om de trekkracht te loggen. De spanning op de katrol wordt geleidelijk opgevoerd. Als de slede in beweging komt wordt deze verder getrokken aan constante snelheid tot de band 0.1 m over de ondergrond is gesleept. Dit opspannen en dan slepen wordt vijf keer herhaald. In de gelogde trekkracht heeft men dan vijf pieken. Het gemiddelde van de vijf pieken is dan de trekkracht voor deze band. De meting is geldig als de vijf metingen binnen de 15 % van hun gemiddelde vallen.

Na het vergelijken van testen met krachtcel en trekweegschaal blijkt de laatste optie snel herhaalbare resultaten op te leveren. Zowel piekkracht als gemiddelde kracht kan gebruikt worden als maatstaaf om banden te vergelijken. Hieronder een voorbeeld van drie slipkracht metingen waarbij de kracht gemeten is met een krachtcel en de data gelogged met Arduino.

Meten Redundantie

Er wordt een score berekend op strafpunten, rekening houdend met het relatief belang van onderstaande observaties:

  • Is, bij gebruik van de fiets, een manier beschikbaar om een licht lossende band bij te pompen?
  • Is, bij gebruik van de fiets, een manier beschikbaar om een lekke band snel te dichten, en op zo'n manier dat de oorspronkelijke bandeigenschappen behouden blijven?
  • Is, bij gebruik van de fiets, een manier beschikbaar om een lekke band snel te vervangen?
  • Hoeveel plaats is er tussen de spaken om een band op te pompen?

Meten prikweerstand

Een gestandaardiseerde pin van 2 mm diameter wordt in de centerlijn van de band geprikt. De pin is gemonteerd op een krachtcel om de prikkracht te meten.

Om deze test uit te voeren is een pers vereist om de band gecontroleerd te kunnen doorprikken. De band wordt opgepompt tot de aangegeven druk en op de pers geïnstalleerd. De pin wordt aan een lage constante snelheid in de band gebracht. Van het raken van de band tot het doorprikken zitten vijf seconden of meer. Voor elke band wordt deze test 3 maal herhaald en het gemiddelde van de maximale krachten wordt genomen.

De kracht wordt gemeten met een krachtcel tussen pers en pin. De zeer lage spanning van de krachtcel wordt ingelezen met een wheat stone bridge interface

  • Datasheet krachtcell
  • Afmetingen krachtcell
  • Datasheet hx711
  • Hhandleiding hx711

De data van de krachtcel interface wordt ingelezen en omgerekend naar spanning door arduino. Vanuit Arduino wordt het over usb via seriële communicatie doorgestuurd naar de computer waar een Scilab programma de data inleest, omrekent naar Newton en opslaat.

                /* Setup and read load cell connected to arduino via the hx711 brakeout board
                 * Code based on sample for digital weight scale of hx711
                 * library design: Weihong Guan (@aguegu)
                 * library host on
                 *https://github.com/aguegu/ardulibs/tree/3cdb78f3727d9682f7fd22156604fc1e4edd75d1/hx711
                 */
                #include <Hx711.h>
                // Make a Hx711 object named scale
                // Hx711.DOUT - pin #A2
                // Hx711.SCK - pin #A3
                Hx711 scale(A2, A3);
                float max = 16777216;
                float druk = 0;

                void setup() {
                  // Setup serial communication
                  Serial.begin(9600);
                  
                  // Set scalefactor for conversion to kg
                  // (uitgang 24 bit ADC -> voltage -> kg
                  scale.setScale(1);
                  //scale.setScale(1992.0/0.033);
                  
                  // Set size of moving average on data to reduce noise
                  //scale.averageValue(3);
                }

                void loop() {
                  float time = millis()/1000.0;
                  
                  // Print weight in kg (NOT IN gram) to serial port
                  // Voor drukkracht max - getGram()
                  druk = scale.getGram();
                  if (druk > max/2.0) druk = max - druk; else druk = - druk;
                  druk = druk / 1000.0; // druk in mV, in kg
                  Serial.print(druk, 3);
                  Serial.print("\t");
                  // Pint a timestamp
                  Serial.println(time, 3);
                  
                  delay(100);
                }
